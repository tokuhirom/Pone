# math

Package math provides basic constants and mathematical functions.

## `sub math.abs($n)`

`abs` returns the absolute value of x.

Call `atan2(3)`.

## `sub math.cos($x)`

Call `cos(3)`.

## `sub math.sin($x)`

Call `sin(3)`.

## `sub math.exp($x)`

Call `exp(3)`.

## `sub math.log($x)`

Call `log(3)`.

## `sub math.sqrt($x)`

Call `sqrt(3)`.
